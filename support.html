How to Use ToolGhor

Complete guide for all file processing tools

Getting Started

ToolGhor processes files entirely in your browser - no uploads, complete privacy.

1. Upload Files

Drag & drop files or click "choose files" to add them to the queue.

2. Select Tool

Choose the appropriate tool based on your file type and desired output.

3. Process

Click the tool button and wait for processing to complete.

4. Download

Processed files download automatically to your device.

PDF Tools

Merge PDFs

Combines multiple PDF files into one document in upload order.

Images → PDF

Converts PNG/JPG images to a single PDF document.

PDF → Images

Extracts first page of each PDF as PNG image.

Rotate PDF

Rotates all pages by 90°, 180°, or 270°.

Delete Pages

Remove specific pages. Format: "1,3,5" or "2-4" for ranges.

Reorder Pages

Rearrange page order. Format: "3,1,2,4" for new sequence.

Image Tools

Convert Format

Convert between PNG, JPG, and WebP formats.

Extract Text (OCR)

Extract text from images and PDFs using optical character recognition.

Video Tools

Convert Video

Convert between MP4, WebM, and AVI formats.

Trim Video

Cut video segments. Use format "HH:MM:SS" or seconds for start time and duration.

Archive Tools

Create ZIP

Bundle all queued files into a single ZIP archive.

Extract ZIP

Unpack ZIP files and download individual files separately.

Privacy & Security

100% Client-Side: All processing happens in your browser

No Uploads: Files never leave your device

No Storage: Files aren't stored on any server

Open Source: Full code transparency

Browser Requirements

Chrome/Edge 90+ • Firefox 88+ • Safari 14+

JavaScript must be enabled for all functionality
